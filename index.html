About the Memory Test Game

The Memory Test is a fun and interactive game designed to improve your memory and cognitive skills. With 20 exciting levels, this game challenges you to memorize numbers of increasing length and input them correctly to score points.

How to Play

  • Each level displays a number for 5 seconds. The number increases in length as you progress, starting from 1 digit and going up to 20 digits.
  • Memorize the number before the timer runs out.
  • After the timer ends, input the number into the provided box and submit your answer.
  • Your performance will be tracked across 20 levels, with each level worth increasing points.

Scoring System

Earn points based on the level you complete:

  • Level 1: 5 points
  • Level 2: 10 points
  • Level 20: 100 points

The maximum score is 1050 points. Achieve 840 points or more to earn an \"Excellent\" rating!
